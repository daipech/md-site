# 欢迎来到主页

这是一个使用 SvelteKit 构建的 Markdown 静态站点。

## 主要功能

本站点提供了以下功能：

- **Markdown 渲染**：完整支持 Markdown 语法
- **代码高亮**：使用 highlight.js 提供优雅的代码显示
- **Mermaid 图表**：支持流程图、序列图等多种图表类型
- **响应式设计**：完美适配桌面端和移动端
- **极简设计**：专注内容，提供优质阅读体验

## 快速开始

访问 [/sample](/sample) 查看完整的 Markdown 功能示例。

访问 [/blog/sample](/blog/sample) 查看动态路由示例。

## 技术栈

本站点使用以下技术构建：

- **SvelteKit** - 现代化的 Web 框架
- **Tailwind CSS** - 实用的 CSS 框架
- **marked** - Markdown 解析器
- **highlight.js** - 代码高亮库
- **mermaid** - 图表生成库
